# 使徒行传｜1 - 8 章


### 使徒行传第一章

#### 1.1. 序言、应许、与耶稣升天( 1 - 11 )

【徒一 3】「祂受害之后，用许多的凭据将自己活活地显给使徒看，四十天之久向他们显现，讲说神国的事。」

- 主耶稣却用「四十天之久」显明复活的事实，「给万人作可信的凭据」（徒十七 31）。
- 主耶稣开始事奉的时候，「四十天受魔鬼的试探」（路四 1）；结束事奉的时候，又用「四十天之久」向门徒显现。两个「四十天」，都是为了败坏仇敌的工作。

【徒一 4】「耶稣和他们聚集的时候，嘱咐他们说：『不要离开耶路撒冷，要等候父所应许的，就是你们听见我说过的。」
