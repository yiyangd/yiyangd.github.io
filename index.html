耶稣教导众人 12.1. 防备假冒为善（1 - 3） 【路十二 1】「这时，有几万人聚集，甚至彼此践踏。耶稣开讲，先对门徒说：『你们要防备法利赛人的酵，就是假冒为善。」 【路十二 2】「掩盖的事没有不露出来的。隐藏的事没有不被人知道的。」 「假冒为善」的意思是「伪装、演戏」，原文用来称赞戴着面具的演员表演得惟妙惟肖。 一个「假冒为善」的人，在生活中也戴着假面具，只追求敬虔的外表、内心却没有敬畏和悔改，人生如戏。 「假冒为善」是一种具有负面感染力的「酵」，当一部分人表演得很「属灵」的时候，其他的人也会跟着装假。人一旦开始装假，就会陷入名利和谎言的网罗。 「防备法利赛人的酵」，连一点点的属灵装假也要马上弃绝 「因为那真正拜父的，要用心灵和诚实拜祂」（约四 23）。 问题： 你能感受到自己的虚伪吗？ 12.2. 不要惧怕，要敬畏 【路十二 4】「『我的朋友，我对你们说，那杀身体以后不能再做什么的，不要怕他们。」 撒旦和世界最多只能杀害身体，但在「杀身体以后不能再做什么」（4 节）。因此，我们「不要怕他们」（4 节） 因为「那些一生因怕死而为奴仆的人」（来二 15），最后还是逃不过身体死亡的结局，一生却白白做了奴仆。 【路十二 5】「我要指示你们当怕的是谁：当怕那杀了以后又有权柄丢在地狱里的。我实在告诉你们，正要怕祂。」 12.3. 在人的面前承认基督（圣灵的工作 9 - 12） 【路十二 10】「凡说话干犯人子的，还可得赦免；惟独亵渎圣灵的，总不得赦免。」 「亵渎圣灵」（10 节），就是「消灭圣灵的感动」（帖前五 19），甚至故意毁谤主耶稣是靠鬼王赶鬼（十一 15），所以「总不得赦免」（10 节）。这是新约圣经中最严肃的宣告与警告。 【路十二 11】「人带你们到会堂，并官府和有权柄的人面前，不要思虑怎么分诉，说什么话；」 【路十二 12】「因为正在那时候，圣灵要指教你们当说的话。』」 问题： 如果“圣灵要指教我们当说的话”，我们能做什么呢？我最近总觉得我除了倚靠，什么都不能做 不可凭着自己的血气说话，因为圣灵会带领他们怎么说、说什么，好在逼迫中能站立得住，勇敢地在人前承认主（11-12 节）。 人若自己强要出头，圣灵就没有机会说话；人若管住自己的口，圣灵就会显明祂的话（12 节）。 一个人越是对自己的心志有把握，越是在十字架面前不堪一击；一个人越是对自己的刚强有信心，越会被十字架打得粉碎。因为十字架的功用，就是破碎人的自信、拆毁人的自傲，完全用主自己来代替。 「注解」预备迎接逼迫的正确做法：

Yiyang Dong published on 2022-02-28
Transfer Learning with MobileNets Many machine learning methods work well ONLY under a common assumption: the training and test data are drawn from the same feature space and the same distribution. When the distribution changes, most statistical models need to be rebuilt from scratch using newly collected training data. In many realworld applications, it is expensive or impossible. In such cases, Transfer Learning between task domains would be desirable. This note illustrate steps of using transfer learning on a pre-trained CNN to build an Alpaca/Not Alpaca （羊驼识别） classifier!
